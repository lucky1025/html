# 报价单生成器

https://lucky1025.github.io/html/

一个完全基于前端的横版报价单制作工具，支持富文本编辑、5 套固定模板 + Auto 智能提色、一键导出 2560×1440 高清 PNG。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建
npm run build

# 预览构建结果
npm run preview
```

## 功能特性

### 编辑能力
| 字段 | 类型 | 说明 |
|------|------|------|
| 标题 | 纯文本输入 | 报价单标题，居中大字展示 |
| 抬头公司名称 | 纯文本输入 | 客户或甲方公司名称 |
| 名称列 | **富文本** | 支持加粗/斜体/下划线/字色/字号/行高/对齐/换行 |
| 内容功能列 | **富文本** | 同名称列 |
| 底部说明 | **富文本（单行）** | 最多 30 字，禁止回车换行，超限自动截断并轻提示 |
| 原价格 | 数字输入 | 仅允许数字和小数点，失焦后格式化（2位小数） |
| 优惠价格 | 数字输入 | 同原价格 |
| 数量 | 数字输入 | 同原价格 |
| 总计 | **自动计算** | = 优惠价格 × 数量（空值按 0 处理），左对齐 |
| 序号 | **自动序号** | 不可手动编辑，随行顺序自动更新 |

### 行操作
- 每行末尾"⋯"菜单：**复制行 / 上移 / 下移 / 删除行**
- PC 端支持**拖拽排序**（拖动序号列的 ⋮⋮ 标识）
- 底部"+ 新增一行"按钮

### 撤销 / 重做
- 快捷键：`Ctrl/Cmd+Z`（撤销）、`Ctrl/Cmd+Y` 或 `Shift+Z`（重做）
- 最多保存 50 步历史

## 模板系统

### 固定模板（5套）

| ID | 风格描述 |
|----|---------|
| T01 | 深蓝专业风，深色表头，浅灰交替行 |
| T12 | 暗紫优雅风，渐变背景，白色卡片 |
| N02 | 森林绿清新风，浅绿背景，绿色渐变表头 |
| N17 | 金橙商务风，暖橙渐变表头，白底卡片 |
| N20 | 深色科技风，全深背景，霓虹绿高亮 |

模板不可配置，只能选择。缩略图使用真实渲染缩放生成，保证"所见即所得"。

### Auto 模板（智能提色）

1. 点击顶部"🎨 Auto提色"按钮，上传一张本地图片
2. 纯前端算法自动提取主色、辅色、背景色
3. 推断渐变方向（横向/纵向）、背景明暗、圆角大小
4. 生成 Auto 模板并加入模板列表，可选中使用

**算法说明：**
- 使用 Canvas 读取图片像素，对像素颜色按饱和度/明度过滤噪点
- 取最高频的前 3 种颜色作为调色板（主色、辅色、背景色）
- 比较上下（top→bottom）与左右（left→right）平均颜色差异，取差异更大的方向作为渐变方向
- 背景明度 > 128 判断为亮色背景，文字使用深色；反之使用浅色

## 导出规格

| 项目 | 规格 |
|------|------|
| 格式 | PNG，非透明背景 |
| 尺寸 | 固定 2560×1440 |
| pixelRatio | ≥ 2（高清） |
| 字体等待 | 导出前 `await document.fonts.ready` |
| 导出前处理 | 自动 blur 所有输入、清理富文本选区高亮 |

### iOS / 内置浏览器兼容
- 标准浏览器：自动触发文件下载，显示"保存成功"toast
- iOS Safari / 微信内置浏览器：可能弹出系统级"分享/保存"交互，属正常行为，页面不做额外确认弹窗
- 若下载被拦截：展示极简 fallback 弹窗，提供"长按图片保存"指引与"打开新页查看"按钮

### 内容溢出拦截规则
- 导出前检测内容渲染高度，按比例折算到 2560 宽度下的等比高度
- 若折算后高度 > 1440px：**禁止导出**，显示 toast 提示："内容过多无法导出，请精简内容"

## 字段校验规则

### 数字列（原价格 / 优惠价格 / 数量）
- 输入时：只允许输入 `0-9` 和 `.`，过滤非法字符
- 输入过程中**不强行插入千分位**，避免打断光标
- 失焦后格式化：保留最多 2 位小数，去除多余末尾 0

### 总计列自动计算规则
```
total = discountPrice × quantity
- 任一为空字符串：total 显示为空（不显示"0"）
- 任一为 0：total 显示"0"
- 结果保留 2 位小数，去除末尾多余 0
```

### 底部说明限制
- 最多 30 个字符（汉字按 1 字符计）
- 禁止 Enter 换行（包括 Shift+Enter）
- 超出时：自动截断到 30 字，仅提示一次轻提示"已达最大字数"（2秒后消失）

## 技术栈

- **框架**：React 18 + TypeScript + Vite
- **样式**：Tailwind CSS v4
- **富文本**：Tiptap（@tiptap/react + 扩展集）
- **拖拽排序**：@dnd-kit/core + @dnd-kit/sortable
- **导出**：html-to-image（pixelRatio、字体等待、跨平台下载）
- **提色**：纯 Canvas 算法，无需后端
